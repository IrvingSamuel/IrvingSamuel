## About Me

Backend-focused engineer with a full-stack background, currently working at **Next Opinion** building integrated systems for educational and financial institutions. Experienced in leading development teams, architecting scalable web solutions, and delivering clean, efficient code under agile methodologies.

Previously managed development projects at XTM Digital and served as IT Manager at Colégio Adventista. Also taught programming (Python, C, C++) at IFAL for the Brazilian Informatics Olympiad (OBI).

- Currently pursuing a **B.Sc. in Computer Science** at Universidade Tiradentes (UNIT)
- Passionate about backend architecture, AI integration, and developer tooling
- Open to collaboration on open-source projects and technical consulting
